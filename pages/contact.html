Announcements

  • announce mailing list: To receive news about releases, events and more via email.
  • Twitter: To receive news about releases, events and more via Twitter.

Reporting bugs

Submit bugs to the issues tab located at our Github repository.

Verify that a duplicate of the issue does not already exist!

Reporting security vulnerabilities

Report security vulnerabilities by creating a security advisory. We would be happy to give you credits for your help with making Uyuni more secure.

GitHub discussions

We have asynchronous communication using GitHub Discussions. They replaced the now archived mailing lists (see below).

Gitter

You can use the channels users and devel at Gitter to chat with us.

You can also join the Gitter rooms via Matrix: #uyuni-project_users:gitter.im and #uyuni-project_devel:gitter.im.

Uyuni Community Hours

Every last Friday of the month, 16.00 CET/CEST (Germany time) we meet online to talk about what is new in Uyuni, discuss how to improve and enhance it, answer questions from the community and have a nice time in general.

Agenda and meeting details are sent to the Uyuni mailing lists and published in the Uyuni wiki too: Uyuni Community Hours.

Deprecated resources
